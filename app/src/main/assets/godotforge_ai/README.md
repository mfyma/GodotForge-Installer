# GodotForge AI Addon

This is a minimal Godot 4 `EditorPlugin` starter installed by **GodotForge Installer**.

## Enable it

1. Open the Godot project.
2. Go to **Project > Project Settings > Plugins**.
3. Enable **GodotForge AI**.

The starter addon performs no network requests and contains no API keys. Add AI-provider integration separately and store secrets outside source control.
